## Features

- **Secure Calculation Engine:** Uses a custom **Shunting-yard algorithm** parser instead of `eval()`, ensuring secure and accurate processing of mathematical expressions.
- **Advanced Arithmetic:** Supports addition, subtraction, multiplication, division, and decimal handling.
- **Smart Input Management:** Includes `AC` (All Clear) and `DEL` (Backspace) functionalities for easy correction.
- **Full Keyboard Support:** Fully accessible via keyboard for a native desktop application feel.
- **Responsive Design:** Optimized layout that looks great on both mobile and desktop devices.
- **Error Handling:** Gracefully manages errors such as division by zero or invalid inputs.

## Technologies

- **HTML5:** Semantic markup and accessible structure.
- **CSS3:** Modern styling with CSS Variables, Flexbox, and Grid layout. Dark mode theme integration.
- **JavaScript (ES6+):** Object-Oriented structure, custom tokenizer, and RPN evaluator.

## Installation & Usage

This project is a static web application and requires no complex installation or dependencies.

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/basic-calculator.git
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd basic-calculator
    ```

3.  **Run the app:**
    - Simply open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, etc.).
